This file contains small "how-to's" we found useful during the development of PPTAM

- How to install a python package for a specific python version? python -m pip install docker
- How to enable the Docker remote API? https://medium.com/@ssmak/how-to-enable-docker-remote-api-on-docker-host-7b73bd3278c6
- How to exclude servers for a proxy? Add no_proxy=localhost,127.0.0.0,127.0.1.1,127.0.1.1,local.home to /etc/environment
